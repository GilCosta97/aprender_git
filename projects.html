My First Project

Employees Management App

A web application for managing employees. We did this as a group for a final project for a class of C#, using this language and Windows Forms.
It was challenging but really fun to do!
The main objective was to make an app that would allow a company to manage their employees, regarding payments, hours of work and area of work, we also added a tab for the official documents needed for recruitment and a contract tab. It has many funcionalities and a simple layout that facilitates navigation and the actual work.

Technologies used include:

  • C#
  • Windows Forms

Ovidio Terraza Website

A website for a restaurant/bar

This was my first real project. I noticed a place i usually go didn't have a website, so i contacted them i proposed to make it.
The project came as a very good experience, in learning and understanding how the real world works between client and developer.
I've made the whole project by myself and today its up and running!
The website has a main page, menu page, reservations page, events and contacts page, with the possibility to change languages between English, French and Portuguese.

Technologies used include:

  • Laravel PHP
  • HTML
  • CSS
  • JavaScript
